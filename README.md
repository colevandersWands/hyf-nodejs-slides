## Installation

This is a typical Node.js app using [Express](https://expressjs.com/) and [PDFKit](https://pdfkit.org/). To install:

```
git clone https://github.com/fdb/hyf-nodejs-slides.git
cd hyf-nodejs-slides
yarn
yarn run dev
```
